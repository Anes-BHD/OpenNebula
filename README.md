# OpenNebula
setting up an open source cloud infrastructure based on OpenNebula, a free and powerful alternative to proprietary solutions.

Requirements:
- Vagrant
- VirtualBox

Usage:

git clone https://github.com/Anes-BHD/OpenNebula.git

cd OpenNebula

vagrant up --provision
